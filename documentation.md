# Installation 
 - installer ionic 

 ```bash
 npm install -g @ionic/cli
 ```

# Creation du projet 
    - creer un projet ionic 
    
```bash

ionic start my-app-ionic blank --type=react

ionic start my-ionic-app sidemenu --type=react

```

# Lancer le projet 
```bash
ionic serve
```

# Structure des pages

#### Les composants sont disponible sur : https://ionicframework.com/docs/components

- 'IonPage' : composant racine de chaque page

- 'IonHeader' : entete de la page

- 'IonToolbar' : barre d'outils

- 'IonContent' : contenu principal de la page

- 'IonTitle' : titre de la page

- 'IonList' : liste d'elements

- 'IonItem' : element de la liste
